Roblox Game Analytics Tracker

This dashboard displays live analytics for the Roblox games you’ve configured in config.js. Data is collected periodically and stored locally; the graphs below visualise the history of visits and concurrent players, along with the percentage growth between successive samples.

Data is polled from Roblox’s Games API using the endpoint documented on the developer forum【698823226887793†L118-L125】 and recorded on your server. Growth percentages are calculated relative to the previous sample.
